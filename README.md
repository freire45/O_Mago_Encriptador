# O_Mago_Encriptador
 
 Programa que Encripta Uma Mensagem

 - Recebe Uma Mensagem Normal e Encripta( Embaralha Ela )
 - Recebe Uma Mensagem Encriptada e Decodifica Ela ( Desembaralha Ela)
 - Capaz de Copiar o Texto Encriptado ou Decodificado
